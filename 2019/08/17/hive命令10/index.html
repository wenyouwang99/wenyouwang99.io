1.创建结构相同的表
create table table_name like table_name1;
语法解释
table_name –要创建表的名字
table_name1 –hive中已有表的名字

2.案例
目标：使用person表的结构创建一个名为person40的表
use census;

create table person40 like person;

select * from person40;

插入数据（能插入就成功）
insert into table person40 values (0,’Bob’,’Burger’),(1,’Charlie’,’clown’);
select * from person40; //此时有2行数据

3.连接
hive支持表之间的等值连接，使你能够整合来自两个表的数据
select table_fields
from table_one
join table_two
on (table_one.key_one = table_two.key_one
and table_one.key_two = table_two.key_two);
语法解释
(1)select table_fields –用于从两个表中选取一系列字段的关键字
(2)from table_one join table_two –罗列出两个为了检索table_fields而进行连接操作的表
(3)on (table_one.key_one = table_two.key_one —列出连接两个表的等值规则
and table_one.key_two = table_two.key_two)

4.案例
目标：在表census.personname和表census.address之间创建连接
use census;

create table census.personname(
persid int,
firstname string,
lastname string
)
clustered by (persid) into 1 buckets
stored as orc
tblproperties(‘transactional’ = ‘true’);

insert into table census.personname
values
(0,’Albert’,’ape’),
(1,’Bob’,’Burger’)
(2,’Charlie’,’Clown’),
(3,’Danny’,’Drywer’);

create table census.address(
persid int,
postname string
)
clustered by (persid) into 1 buckets
stored as orc
tblproperties(‘transactional’ = ‘true’);

insert into table census.address
values
(1,’KA13’),
(2,’KA9’),
(10,’SW1’);

现在执行连接操作
select
personname.firstname,
personname.lastname,
address.postname
from
census.personname
join
census.address
on
(personname.persid = address.persid);

结果如下
Bob Burger KA13
Charlie Clown KA9

5.使用外连接
hive支持采用left,right,full outer等连接方式实现表之间的等值连接，这其中无匹配的键
select table_fields
from table_one
[left,right,full outer]
join table_two
on (table_one.key_one = table_two.key_one
and table_one.key_two = table_two.key_two);
语法解释
(1)left –left 连接产生的结果包含表table_one中匹配where语句的字段值以及表
table_two中匹配和不匹配where语句的字段值
(2)right –right 连接产生的结果包含表table_one中匹配where语句的字段值以及表
table_one中匹配和不匹配where语句的字段值
(3)full outer –full outer 连接将返回表table_two和table_one中的字段值；当有不匹配
where语句的行时，其字段值为null
(4)on –列出两个表的等值规则

6.案例
左连接方式
use census;

select
personname.firstname,
personname.lastname,
address.postname
from
census.personname
left join
census.address
on
(personname.persid = address.persid);

此时结果是4条记录
Albert Ape null
Bob Burger KA13
Charlie Clown KA9
Danny Drywer null

右连接方式
use census;

select
personname.firstname,
personname.lastname,
address.postname
from
census.personname
right join
census.address
on
(personname.persid = address.persid);

此时结果是3条记录
Bob Burger KA13
Charlie Clown KA9
Danny Drywer null

使用完全外连接
use census;

select
personname.firstname,
personname.lastname,
address.postname
from
census.personname
full outer join
census.address
on
(personname.persid = address.persid);

此时结果是5条记录
Albert Ape null
Bob Burger KA13
Charlie Clown KA9
Danny Drywer null
null null SW1
